# ponderada_05_BMO

## Parte 1: Montagem Física e Programação com LDR e Modo Noturno


Nesta etapa, vocês deverão montar o semáforo e conectar o sensor LDR, permitindo que ele detecte a passagem de carros simulados por variação de luz. O comportamento do semáforo deve mudar para o modo noturno quando a luminosidade cair. Documente o funcionamento do LDR e como ele está integrado ao sistema.



## Parte 2: Configuração da Interface Online


Nesta parte, vocês vão criar uma interface online que permita ajustar o comportamento do semáforo e ativar o modo noturno, além de visualizar os dados captados pelo LDR. A interface deve ser simples e funcional, facilitando a interação com o semáforo.



## Parte 3: Comunicação entre Semáforos


Aqui, o verdadeiro desafio será fazer com que os semáforos do grupo se "comuniquem", formando uma rede que regula o tráfego de forma inteligente em uma cidade. Explore como essa comunicação pode melhorar o gerenciamento do fluxo de veículos.


⚠️ Entrega
Poste no GitHub a montagem física, o código do semáforo, o link da interface online e um vídeo demonstrando o funcionamento e a comunicação entre os semáforos do grupo.
